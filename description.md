# VCFormer 量化选股回测框架


基于变量维度注意力与时序模式注意力的多任务排序学习选股框架。

## 主要特性

- **变量维度注意力（VCformer）**  
  在因子维度计算自注意力，捕捉动量、波动率、换手率等特征之间的横截面交互关系。

- **时序模式注意力（TPA）**  
  通过多尺度一维卷积提取局部时序形态，并利用注意力机制自动筛选关键模式，无需人工定义窗口长度。

- **排序学习优化**  
  使用 Pairwise Hinge Loss 直接优化股票间的相对排序，与信息系数（IC）优化方向一致，避免预测绝对收益带来的噪声问题。

- **多任务学习**  
  联合优化排序损失、波动率预测损失和时序分解损失，提升共享表示的稳健性与泛化能力。

- **模块化与可配置**  
  数据清洗、因子构建、模型训练、评估完全解耦；所有超参数通过统一配置文件（`config.py`）集中管理，便于调整股票池规模、序列长度等设置。

- **损失与预测监控**  
  提供训练/验证损失、预测得分分布统计，辅助分析模型收敛行为与泛化表现。

## 🖥️ 使用示例

### 1. 下载并清洗数据
```bash
python data/download.py --start 2018-01-01 --end 2025-06-30
python data/clean.py
```

```bash
python main.py --config config.py --epochs 50 --batch_size 256
```

```bash
python backtest_main.py --model_path ./saved_models/best.pth --output ./results
```

## 实验数据与结果

### 数据范围与实验设置

- **股票池**：A股市场流动性较高的25只股票（调试阶段），包含平安银行、万科A、浦发银行等
- **回测区间**：2023年7月18日至2026年7月17日（约3年，697个交易日）
- **特征维度**：50+ 量价因子（动量、波动率、流动性、技术指标、时序分解、动态波动率因子）
- **模型配置**：VCformer + TPA，序列长度30天，批次大小256，训练轮次20
- **训练/测试划分**：时间顺序划分，单次训练（非滚动窗口）
- **交易成本**：双边0.15%费率 + 0.05%滑点
